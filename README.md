# Movies Server

This repository contains rest apis for movies db

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file

`DB_USER` ,
`DB_PASS` ,
`DB_NAME` ,
`DB_HOST` ,
`PORT`,
`DB_DRIVER`

For example, please refer the .env.example file in the root directory

## Installation

Install onito_backend with npm

```bash
  npm install onito_backend
  cd onito_backend
```
## Run Locally

Clone the project

```bash
  git clone https://github.com/AswathyRajM/onito_backend.git
```

Go to the project directory

```bash
  cd onito_backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start:dev
```

```bash

```

Install dependencies

```bash
  npm install
```

Run the migrations

```bash
  npx sequelize-cli db:migrate
```

Run the seed files

```bash
  npx sequelize-cli db:seed:all
```

Start the server

```bash
  npm run start:dev
```
